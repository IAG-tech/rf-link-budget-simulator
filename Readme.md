# RF Link Budget Simulator

## Overview

A small RF simulation toolkit written in GNU Octave to explore radio propagation, link budget and coverage estimation.

## Features

- link budget calculation
- propagation models
- shadow fading

## Implementations

- Octave prototype
- Python modular version (WIP)
